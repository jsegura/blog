Websockets Posted on May 13, 2014
What are websockets?
websockets

A Websocket is a protocol that provides dual-channel communication between web servers and web browsers. It uses the HTTP protocol as a handshake but it has its own TCP-based protocol. This is useful for enabling a better interaction between your web users, it’s a real-timeand once the request is started by the client, the server can send information over the websocket without the requisition from the client. A demo site Echo test is available using ws://echo.websocket.org.

hugo blog
